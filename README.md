# Tetris&Breakout

## 開発環境
- Visual Studio 2013
- Visual Studio 2015

## 必要動作環境
- Windows98以降のOS
- DirectX9以降のインストール
- .NET Framework 4.0以降のインストール

- ゲームの実行ファイルとDxLib.dllを同じフォルダに入れること。
	- DxLib.dllは下記配布サイトより入手可能。
	- またこのリポジトリの「実行ファイル」フォルダにも配布されているのと同じのがあるため使用可。
	- http://dxlib.o.oo7.jp/dxdload.html
