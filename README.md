- 概要
	- マリオの生みの親である宮本 茂氏曰く、簡単にできる操作を「２つ同時にやろう」とすると意外と難しく、失敗すると悔しい。そんなところにゲームの面白さがあるとか、なんとか。ほんとかな？試してみよう。 テトリスとブロック崩しで。
	- ”実行ファイル”ディレクトリにある”DXlib_CS.exe”を実行することでゲーム開始
	- Trainingモードでは練習ができる。このモードではユーザのレベルに合わせて難易度が変わるため、丁度いい難易度で遊ぶことができる。
		- Trainingモードにおける難易度はAIによって管理されている。このAIはユーザの緊張度を計測し、ユーザが緊張しているなら難易度を下げる、ユーザがリラックスしているなら難易度を上げるといった処理を行う。
		- 緊張度の計測は”テトリスのミノを回転させている＝プレイヤーに余裕がない＝緊張している”や”ブロック崩しのボールをバーの中心で受け止めている＝余裕がある＝リラックスしている”といった実装を行っている。
- 必要動作環境
	- Windows98以降のOS
	- DirectX9以降のインストール
	- .NET Framework 4.0以降のインストール
	- ゲームの実行ファイルとDxLib.dll,Difficulty.iniを同じフォルダに入れること。
